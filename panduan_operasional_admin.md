# 📘 Panduan Operasional Admin E-Katalog PLS

Panduan ini berisi cara menggunakan Dashboard Admin untuk memperbarui data website secara mandiri.

---

## 1. Cara Mengakses Dashboard
1. Buka alamat website Anda dan tambahkan `/admin.html` di akhir URL (Contoh: `katalog-pls.com/admin.html`).
2. Masukkan **Email** dan **Password** Admin yang telah didaftarkan.
3. Klik **Masuk**.

---

## 2. Mengelola Sektor & Jabatan Karier
Menu ini digunakan untuk mengatur "Pohon Karier" di halaman depan.
*   **Tambah Jabatan:** Klik tombol **"+ Tambah Jabatan"**. Pilih sektor yang sesuai (Pemerintahan, LSM, dll), masukkan nama jabatan, deskripsi singkat, dan instansi terkait.
*   **Edit Data:** Klik ikon pensil (✏️) pada tabel untuk mengubah informasi jabatan.
*   **Hapus Data:** Klik ikon sampah (🗑) jika jabatan tersebut tidak ingin ditampilkan lagi.

---

## 3. Mengelola Profil Alumni
Menu ini untuk menampilkan kisah sukses lulusan.
*   **Tambah Alumni:** Klik **"+ Tambah Alumni"**. Isi Nama, Tahun Lulus, Pekerjaan, dan Pesan/Quote.
*   **Foto Alumni:** Anda bisa mengunggah foto alumni langsung. Pastikan foto memiliki rasio kotak (1:1) agar terlihat rapi.
*   **LinkedIn:** Anda bisa memasukkan link profil LinkedIn alumni agar bisa dihubungi oleh mahasiswa.

---

## 4. Mengelola Branding & Hero (Halaman Utama)
Gunakan menu ini jika ingin mengubah tampilan awal website.
*   **Judul Utama:** Mengubah teks besar di bagian paling atas website.
*   **Ganti Background:** Klik **"📷 Ganti Background"** untuk mengunggah gambar latar belakang baru pada bagian Hero.
*   **Badge Text:** Mengubah teks kecil di atas judul utama.

---

## 5. Mengelola Kata Pengantar (Kaprodi)
*   **Isi Pesan:** Masukkan kutipan atau sambutan dari Kaprodi.
*   **Foto Kaprodi:** Klik **"📷 Pilih Foto"** untuk memperbarui foto Kaprodi yang baru.

---

## 6. Pengaturan Keamanan & Backup
*   **Ganti Password:** Buka menu **Pengaturan** (⚙️). Masukkan password baru Anda. Disarankan mengganti password secara berkala.
*   **Backup Data (Export):** Sangat disarankan untuk mengklik **"📥 Export JSON"** sebulan sekali. Ini akan mengunduh seluruh data website sebagai cadangan jika terjadi kesalahan.
*   **Restore Data (Import):** Gunakan file cadangan JSON untuk mengembalikan data website ke kondisi sebelumnya.

---

### 💡 Tips Penting:
1. **Selalu Klik Simpan:** Setiap kali Anda melakukan perubahan di panel mana pun, pastikan Anda menekan tombol **"💾 Simpan"** yang berada di pojok kanan atas agar data tersimpan permanen ke server.
2. **Kualitas Gambar:** Gunakan gambar yang tidak terlalu berat (disarankan di bawah 1MB) agar website tetap cepat saat dibuka.
